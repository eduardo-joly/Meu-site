# 🧠 Quiz Interativo para Programadores
Uma aplicação web moderna, interativa e responsiva desenvolvida para testar os conhecimentos de desenvolvedores em conceitos fundamentais de desenvolvimento web e programação.
---
## 🚀 Funcionalidades
- **Avaliação em Tempo Real**: O quiz processa as respostas localmente com JavaScript e fornece feedback instantâneo para cada questão (indicação de correto/incorreto com explicações explicativas).
- **Barra de Progresso Dinâmica**: Um indicador visual acompanha o preenchimento das perguntas em tempo real.
- **Vários Formatos de Pergunta**: Suporte a múltipla escolha (radio buttons), seleção de múltiplas caixas (checkboxes), entrada de texto, verificação de ano (number), validação de força de senha, envio de arquivos e menus suspensos (selects).
- **Tema Escuro Moderno (Dark Mode)**: Layout premium baseado em Glassmorphism, feito com CSS Vanilla sem frameworks.
- **Responsivo**: Adaptado para telas de smartphones, tablets e desktops.
---
## 🛠️ Tecnologias Utilizadas
- **HTML5**: Estrutura semântica e acessível das seções de perguntas.
- **CSS3 (Vanilla)**: Design moderno, animações de micro-interações, efeitos de gradiente e responsividade (Flexbox/Grid).
- **JavaScript (Vanilla)**: Lógica de pontuação dinâmica, barra de progresso, validação de força de senha (Regex) e exibição do gabarito.
- **Google Fonts**: Tipografia moderna utilizando a fonte *Outfit*.
